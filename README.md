# FenixDuels
Professional duel plugin for Minecraft servers.

## Xususiyatlari
- Moslashuvchan konfiguratsiya
- Kuzatuvchi (Spectator) tizimi
- Arena va Kitlarni oson boshqarish
- Tab-Completer yordami

## Buyruqlar
- `/duel <o'yinchi> <kit> <raundlar>` - Duel taklif qilish
- `/duel accept` - Duelni qabul qilish
- `/spec <o'yinchi>` - O'yinni kuzatish
- `/duel setspawn <arena> <spawn1/2>` - (Admin) Arena spawnini o'rnatish
- `/duel deny` - Duelni bekor qilish
- `/duel createarena <name><kit>` - arena yaratish
- `/duel createkit <name>` - kit yaratish
- `/duel wand` - maxsus arena belgilash tayoqchasi
- `/duel setregion <arena>` - arenaga hududni ulash
- `/duel see <kit>` - kit inventorini ko'rish

## Ruxsatnomalar (Permissions)
- `fenixduels.use` - Asosiy duel buyruqlari
- `fenixduels.admin` - Admin buyruqlari
